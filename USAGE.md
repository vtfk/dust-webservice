## API endpoints

### ``GET /:service/invoke``

```json
{
  "fileName": "ScriptFileName.ps1",
  "args": {
    "SamAccountName": "sak8976",
    "Properties": [
      "title",
      "mail"
    ],
  }
}
```

{

  "args": {

    ],
  }
}
